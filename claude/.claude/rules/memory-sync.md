# Memory Synchronization

At the end of every chunk of work, synchronize the agent's memory to ensure all
updates are saved and available for future reference. This is crucial for
maintaining context across interactions and ensuring that the agent can recall
past information accurately.

Ensure that any CLAUDE.md or MEMORY.md files are updated with the latest
information before proceeding to the next task. This includes any new facts,
changes in state, or relevant details that have been discovered or modified
during the current interaction.
